git init

Initializes a new Git repository

git add

Stages changes for commit

git commit

Records staged changes with a message

git pull

Fetches and merges changes from a remote repository

git push

Pushes local changes to a remote repository.

git branch

Lists or creates branches

git checkout

Switches to a different branch or commit.

git merge

Combines changes from one branch into another

git status

Shows the status of your working directory

git log

Displays a history of commits

git diff

hows the differences between files or commits

git stash

Temporarily saves changes not ready to be committed

git remote

Manages remote repositories

git fetch

Retrieves changes from a remote repository

git reset

Resets the HEAD to a previous commit

git clone

Copies a remote repository to your local machine, creating a local copy of the entire project, including all branches and commit history.
